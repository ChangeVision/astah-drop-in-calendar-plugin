Drop in Calendar plugin for Astah
===============================================

![sample image](https://github.com/ChangeVision/astah-drop-in-calendar-plugin/raw/master/doc/screeenshots/calendar.png)
![sample image_ja](https://github.com/ChangeVision/astah-drop-in-calendar-plugin/raw/master/doc/screeenshots/calendar_ja.png)

Version
------------
1.0.2

Available for
------------
Astah UML, Astah Professional 6.5.x or later

Description
------------
This Plugin enables you to Drops a Calendar instantly into a diagram when you need it - Great when taking a minutes in a meeting or making plans using Mind Map.

How to install
------------
1. Deploy the jar file in the "plugins" folder under the Astah Install folder
2. Start Astah
3. You find that the [Stick Calendar] has been added under the [Tool] menu

![menu image](https://github.com/ChangeVision/astah-drop-in-calendar-plugin/raw/master/doc/screeenshots/calendar_menu.png)
