Stick Calendar plugin for Astah
===============================================

![sample image](https://github.com/ChangeVision/astah-stick-calendar-image-pluginraw/master/doc/screenshots/calendar.png)

Version
------------
1.0.0

Available for
------------
Astah UML, Astah Professional 6.5.x or later

Description
------------
This Plugin enables you to stick calendar into Astah diagrams.

How to install
------------
1. Deploy the jar file in the "plugins" folder under the Astah Install folder
2. Start Astah
3. You find that the [Stick Calendar] has been added under the [Tool] - [export image] menu
